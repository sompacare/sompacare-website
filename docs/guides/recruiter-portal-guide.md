# Recruiter Portal — User Guide

**URL:** https://recruiter.yourdomain.com (local: http://localhost:3003)

For recruiters to manage candidate pipelines, schedule interviews, send offers, and track placements.

---

## Pipeline board

Kanban view of candidates by stage:

`APPLIED` → `SCREENING` → `INTERVIEW` → `OFFER` → `PLACED`

Drag candidates between stages or update from candidate detail page.

---

## Candidate detail

Per-candidate actions:

| Action | Description |
|--------|-------------|
| Parse resume | AI extracts skills, experience, credentials |
| Schedule interview | Set date, mode (phone/video/in-person) |
| Send offer | Specify facility, pay rate, start date |
| Accept offer | Mark candidate acceptance |
| Send onboarding | Trigger onboarding package |
| Update checklist | Background check + reference status |

---

## Leaderboard

Recruiter rankings by placement count. Use for team motivation and performance reviews.

---

## Metrics (home)

- Active pipeline count
- Total placements
- Breakdown by stage
- Recent placements with facility names

---

## Tips

- Move candidates through stages promptly — stale pipelines reduce fill rates
- Use AI resume parse to pre-fill clinical role and match scores
- Link candidates to facilities early for faster placement
